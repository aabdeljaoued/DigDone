# DigDone

Recurring memo reminders for Android.

## Stack

- Kotlin
- Jetpack Compose
- DataStore
- Kotlinx Serialization
- AlarmManager + Boot receiver

## Features

- Create recurring memos: hourly, daily, weekly, monthly, yearly
- Trigger notifications when a memo is due
- Disable notifications with a phrase-based setting
- Persist memo data locally
- Reschedule after reboot

## Notes

This repository is scaffolded for Android Studio/Gradle usage. Add the Android SDK and Gradle wrapper locally if you want to build it on a machine with the Android toolchain installed.
